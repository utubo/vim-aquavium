*aquavium.txt*  Plugin to Show 🐟!

Functions:

aquavium#Show([{option}])
	Show 🐟 in the current window.

aquavium#TabPanel([{option}]): string
	You can set this to 'tabpanel'
	e.g. `set tabpanel=%!aquavium#TabPanel()`

aquavium#TabPanelPart([{option}]): string
	You can use this in your tabpanel function.

{option} is dict.
	height:	The height. Default is window height.
	width:	The width. Default is window width.
